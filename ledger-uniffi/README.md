# Ledger UniFFI

UniFFI bindings for the Midnight Ledger crate.

## Quick Start

### Android
```bash
./run.sh android
cd rn-demo-app
npx react-native run-android
```

### iOS
```bash
./run.sh ios
cd rn-demo-app
npx react-native run-ios
```

### Both Platforms
```bash
./run.sh
```

## Prerequisites

- **Android**: Android NDK installed
- **iOS**: Xcode installed
- **Rust**: Latest stable Rust toolchain
- **Node.js**: For React Native development
